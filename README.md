# Chat em Tempo Real

Projeto fullstack com:

- NestJS (backend)
- Next.js (frontend)
- Socket.IO (WebSocket)
- Prisma (banco de dados)

## Funcionalidades
- Chat em tempo real
- Histórico de mensagens
- Indicador de presença
- Estilização por usuário

## Como rodar

### Backend
- cd back
- npm install
- npm run start:dev

### Frontend
- cd front
- npm install
- npm run dev
